Features

Code Editor

Multi-tab editor with syntax highlighting for Python, JavaScript, HTML, CSS, JSON, and more. Line numbers, auto-indent, bracket matching.

Real Terminal

Full interactive shell — not fake commands. Auto-detects Bash, PowerShell, CMD, Zsh, Fish. Command history, Ctrl+C interrupt.

Git Integration

Clone, init, commit, push, pull, diff, log, branches. Auto-detects repos. Prompts to install Git if missing.

Extensions

Browse the VS Code Marketplace. Search, install, and manage extensions directly from the sidebar.

Command Palette

Ctrl+Shift+P opens 50+ fuzzy-searchable commands. Go to line, toggle comment, switch theme, and more.

Settings GUI

Visual settings panel with live preview. Change font, theme, tab size, word wrap. Searchable and resettable.

Themes

Beautiful dark and light themes inspired by Catppuccin. Custom SVG icons. Round modern buttons with hover effects.

Auto Updates

Checks GitHub for new releases on startup. Shows changelog, version comparison, and platform-specific download.
