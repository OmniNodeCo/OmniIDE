Features

Code Editor

Multi-tab editor with QPlainTextEdit, syntax highlighting via QSyntaxHighlighter, line numbers, auto-indent, current line highlight, and bracket matching.

Real Terminal

Full interactive shell via QProcess. Auto-detects Bash, PowerShell, CMD, Zsh, Fish. Command history, restart, and clear.

Git Integration

Clone, init, commit, push, pull, diff, log, branches, set remote. Auto-detects repos. Prompts to install Git if missing.

VS Code Extensions

Browse the VS Code Marketplace. Search, download, and install VSIX packages. Parses package.json for contributes metadata.

Command Palette

Ctrl+Shift+P opens a fuzzy-searchable list of all commands. Quick access to every feature without touching the mouse.

Settings GUI

Visual settings dialog with dropdowns, spinners, and checkboxes. Live-apply font, theme, tab size, word wrap, and more.

Themes

Dark and light themes with Catppuccin-inspired colors. Full QSS stylesheet system with 200+ styled widget rules.

Auto Updates

Checks GitHub Releases on startup. Shows version comparison and opens platform-specific download page.
