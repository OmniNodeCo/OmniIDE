Features

Code Editor

QPlainTextEdit with QSyntaxHighlighter, line numbers, auto-indent, current line highlight.

Real Terminal

QProcess interactive shell. Bash, PowerShell, CMD, Zsh, Fish. History and restart.

Git Integration

Clone, commit, push, pull, diff, log, branches. Auto-detect repos. Git install prompt.

VS Code Extensions

Browse marketplace. Download VSIX packages. Parse package.json for metadata.

Command Palette

Ctrl+Shift+P fuzzy search all commands. Quick access to everything.

Settings GUI

Visual dialog with dropdowns, spinners, checkboxes. Live-apply everything.

Themes

Dark and light Catppuccin themes. Full QSS stylesheet with 200+ rules.

Auto Updates

Checks GitHub Releases on startup. Version comparison and download link.
