# 🚀 OmniIDE by OmniNodeCo

A fast, modern, lightweight desktop IDE built from scratch.
No Electron. No bloat. Pure speed.

## Features

- 🎨 Modern dark/light themes
- 📁 File tree explorer
- ✏️ Multi-tab code editor
- 🎯 Syntax highlighting (Python, JS, HTML, CSS, JSON)
- 🔍 Find & Replace
- 💻 Built-in terminal
- ⌨️ Keyboard shortcuts
- 📂 Recent files tracking
- ⚡ Instant startup

## Installation

```bash
git clone https://github.com/OmniNodeCo/OmniIDE.git
cd OmniIDE
pip install -r requirements.txt
python run.py
Build Executable
Bash

pip install pyinstaller
pyinstaller --onefile --windowed --icon=assets/icon.png --name=OmniIDE run.py
Shortcuts
Action	Shortcut
New File	Ctrl+N
Open File	Ctrl+O
Save	Ctrl+S
Save As	Ctrl+Shift+S
Close Tab	Ctrl+W
Find	Ctrl+F
Toggle Terminal	Ctrl+`
Toggle Sidebar	Ctrl+B
Tech Stack
Python 3.10+
Tkinter + ttkbootstrap (modern UI)
No Electron, No heavy frameworks
